# **App Name**: PetSpa Agendamentos

## Core Features:

- Agendamento: Coleta informações do cliente e pet (nome, raça, porte, contato, vacinação), tipo de banho e adicionais.
- Disponibilidade: Permite agendamento em intervalos de 30 minutos, das 8h às 17h (Brasília), exceto domingos.
- Orçamento: Calcula o valor total com base nos serviços e adicionais selecionados.
- Link de Compartilhamento: Gera um link contendo todas as informações do agendamento para enviar via WhatsApp.

## Style Guidelines:

- Primary color: Pink (#E91E63) to create a vibrant and playful atmosphere.
- Secondary color: Gold (#FFC107) to add a touch of luxury and sophistication.
- Text color: Dark Brown (#212121) for readability against the bright background colors.
- Body and headline font: A playful, slightly rounded font like 'Baloo 2' or similar to complement the cheerful design.
- Use ornate, gold-colored icons depicting pets and grooming elements, mirroring the image's style.
- A balanced layout with a central focus, utilizing decorative elements inspired by the image to frame content sections.
- Subtle, elegant animations, like fading transitions or a gentle shimmer effect on gold elements.
